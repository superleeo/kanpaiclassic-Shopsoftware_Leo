## 🚀 第一步：系统安装

### 1.1 前置要求
- **PHP版本**: 7.4+ (建议 8.0+)
- **数据库**: MySQL 5.7+ 或 MariaDB 10.2+

- **PHP扩展**: MySQLi, PDO, GD, cURL, JSON

### 1.2 安装步骤

1. **访问安装程序**

## 🔧 技术支持

### 常见问题

**Q1: 如何添加商品图片？**
- 在商品编辑页面中，有"图片"选项卡
- 支持PNG、JPG、GIF格式
- 建议分辨率: 800x600px或更高

**Q2: 如何设置特价商品？**
- 在商品编辑中有"特价"字段
- 设置特价百分比或固定金额
- 可设置特价时间段

**Q3: 如何启用会员折扣？**
- **管理后台** > **折扣管理**
- 创建会员等级
- 为不同等级设置折扣

**Q4: 如何处理订单？**
- **管理后台** > **订单管理**
- 接收订单后更新配送状态
- 发送确认和配送邮件
